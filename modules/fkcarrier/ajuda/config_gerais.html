Configurações Gerais

Objetivo

    Definir os parâmetros globais de funcionamento do módulo.

Meu CEP

    Informe o CEP de onde serão despachados os produtos. Este CEP será utilizado como base do cálculo do frete até seu destino.

CEP da minha cidade

    Neste campo deve ser informado os intervalos de CEP de sua cidade.
    Informe o CEP inicial e o CEP final e clique no botão incluir. Os intervalos de CEP serão mostrados no campo logo abaixo em formatação especial.
    Caso necessário, este campo pode ser editado manualmente, no entanto deve ser tomado cuidado com sua formatação.

    Obs: Caso sua cidade seja uma metrópole cujas cidades vizinhas são tratadas como parte integrante de sua cidade, o intervalo de CEP das mesmas também devem ser cadastrados (essas informações podem ser obtidas junto aos Correios de sua cidade).

    Link dos Correios para pesquisa.

Correios

    Neste campo informe:
    • Se deseja que o valor do frete seja calculado considerando somente as tabelas offline de preços. O web service dos Correios não será utilizado, em razão disso os prazos de entrega serão baseados nos dados previamente cadastrados.
    • Marque os serviços adicionais dos Correios que deseja utilizar.
    • Quando marcado algum dos serviços adicionais, deve ser selecionado como os mesmos devem ser calculados:
      • Considerar embalagem: o cálculo será feito por embalagem gerada no pedido, exemplo: se no pedido existirem 3 produtos embalados individualmente os valores dos serviços adicionais serão triplicados. Esta opção deve ser considerada quando os produtos são despachados individualmente nos Correios.
      • Considerar pedido: o cálculo será efetuado considerando o envio de todas as embalagens em pacote único, ou seja, os valores dos serviços adicionais serão únicos e portanto mais baratos.

Frete e Envio

    Neste campo informe:
    • Quantidade em dias que deve ser adicionado ao prazo de entrega dos serviços dos Correios e das transportadoras.
    • Valor a ser adicionado ao valor do frete calculado.
    • Se o valor do frete deve ser calculado com base em Embalagens padrões ou Embalagens individuais. A escolha deve ser baseada no perfil dos produtos comercializados e na quantidade média de produtos por pedido.
      O módulo irá processar o cálculo conforme abaixo:
      • Embalagens padrões: com base no volume cúbico dos produtos, o módulo escolherá a melhor embalagem cadastrada para acondicioná-los. O módulo tentará acondicionar vários produto em uma mesma embalagem para otimizar o valor do frete.
      • Embalagens individuais: o cálculo será feito individualmente por produto.
    • Se além da transportadora definida para o Frete Grátis, as demais transportadora com seus valores de frete devem aparecer como opção para o cliente.
    • Se o cálculo do frete no Carrinho de Compras deve ser efetuado somente com o cliente logado. Esta opção aumentará a performance da loja para os clientes não logados.
      Os blocos de informação do cálculo do frete irão calcular o frete normalmente.

Bloco de informações

    Marque as posições onde deseja que apareça o bloco de solicitação de CEP destino e resumo do valor do frete.

Botão Salvar

    Clique no botão para validar as alterações efetuadas.

Botão Excluir cache

    Utilize este botão para liberar espaço no banco de dados utilizado pelo cache do módulo. Este procedimento não é obrigatório mas dependendo do volume de vendas é aconselhável executá-lo periodicamente a cada 3 meses.
    A limpeza do cache é efetuada automaticamente pelo módulo quando alterada as Especificações dos Correios ou Geradas as Tabelas Offline.
