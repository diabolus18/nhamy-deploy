Configuração do módulo

    Provedores de CEP

      Selecione o serviço que será utilizado para pesquisa online do CEP do cliente. Observe que dentre os serviços disponíveis existem serviços pagos (AutoCep) e serviços gratuitos (República Virtual e BYJG).

    Grupo de Clientes - Pessoa Jurídica

      Mantenha Padrão caso não vá utilizar essa função ou selecione o Grupo de Clientes que deve ser assumido quando um novo cliente Pessoa Jurídica se cadastrar.

    Diversos

    • Marque Preencher automaticamente o campo UF se desejar que o módulo preencha este campo com os dados obtidos da pesquisa do CEP;
    • Marque Verificar duplicidade de CPF e CNPJ se não quiser permitir a abertura de novo cadastro para CPF e CNPJ já cadastrados;
    • Marque Manter os novos campos na desinstalação se desejar que os novos campos criados pelo módulo sejam mantidos em caso de desinstalação do módulo. Essa função é importante caso seja necessário desinstalar/instalar o módulo;

Importante

    Caso deseje instalar um novo tema ou atualizar a versão do Prestashop, proceda conforme abaixo:
    • Marque a opção Manter os novos campos na desinstalação;
    • Desinstale o FKcustomers;
    • Instale o novo tema ou efetue a atualização do Prestashop;
    • Reinstale o módulo FKcustomers.
